# 🏅 运动员技术等级查询助手 使用指南

> **一句话介绍**：这是一个自动帮你查询国家体育总局运动员技术等级证书的小工具，你只需要准备好证书编号和姓名名单，运行程序后它会自动打开浏览器、填写信息、查询结果，并把结果和相关的秩序册/成绩册下载保存到你的电脑上。

---

## 📦 你需要提前准备什么？

在开始之前，请确认你的电脑满足以下条件：

| 项目 | 要求 | 检查方法 |
|------|------|---------|
| **操作系统** | Windows 10/11、macOS 或 Linux | 看电脑属性 |
| **Python** | 版本 3.8 或更高 | 见下方「安装 Python」步骤 |
| **Google Chrome** | 已安装在电脑上 | 桌面上能看到 Chrome 图标 |
| **网络** | 能正常访问 https://zwfw.sport.gov.cn | 用 Chrome 打开这个网址试试 |

---

## 🐍 第一步：安装 Python

如果你不确定电脑上有没有 Python，或者版本够不够，请按以下步骤操作：

### Windows 用户

1. 打开浏览器，访问 https://www.python.org/downloads/
2. 点击页面上最大的黄色按钮 **「Download Python 3.x.x」**
3. 下载完成后，**双击运行** 下载的 `.exe` 文件
4. ⚠️ **非常重要**：在安装窗口底部，勾选 **「Add Python to PATH」**（添加到环境变量），然后再点击 **Install Now**
5. 等待安装完成，点击 Close

### 验证安装是否成功

1. 在键盘上按 `Win + R`，输入 `cmd`，按回车，打开黑色命令窗口
2. 输入以下命令，按回车：
   ```
   python --version
   ```
3. 如果看到类似 `Python 3.11.4` 的输出，说明安装成功 ✅
4. 如果提示「不是内部或外部命令」，说明安装时没勾选 PATH，请重新安装一遍

---

## 📥 第二步：下载本项目文件

你需要下载两个文件放到同一个文件夹里：

1. **`sport_level_crawler.py`** —— 主程序（自动查询脚本）
2. **`input.json`** —— 你要查询的名单（证书编号 + 姓名）

### 操作步骤

1. 在你的电脑上新建一个文件夹，比如放在桌面上，命名为 `证书查询`
2. 把 `sport_level_crawler.py` 文件保存到这个文件夹里
3. 在同一个文件夹里，新建一个文本文件，命名为 `input.json`
4. 用「记事本」（Windows）或「文本编辑」（macOS）打开 `input.json`，粘贴以下内容：

```json
[
  {
    "证书编号": "CFA201900050",
    "姓名": "胡睿宝"
  }
]
```

> 💡 **想查更多人？** 按照上面的格式，在大括号 `}` 后面加一个英文逗号 `,`，然后复制粘贴一整组大括号内容即可。例如：
> ```json
> [
>   {"证书编号": "CFA201900050", "姓名": "胡睿宝"},
>   {"证书编号": "CFA201900051", "姓名": "张三"},
>   {"证书编号": "CFA201900052", "姓名": "李四"}
> ]
> ```

5. 保存并关闭 `input.json`

---

## ⚙️ 第三步：安装程序依赖

程序需要用到两个外部工具库，只需运行一次命令即可安装。

### 操作步骤

1. 打开命令窗口（按 `Win + R`，输入 `cmd`，回车）
2. 用 `cd` 命令进入你刚才创建的文件夹。假设文件夹在桌面上：
   ```
   cd Desktop\证书查询
   ```
   > 如果你把文件夹放在别的地方，比如 D 盘，命令可能是 `cd D:\证书查询`
3. 依次运行以下两条命令（复制粘贴后按回车）：
   ```
   python -m pip install playwright requests
   ```
   等待显示 `Successfully installed` 字样。

   ⚠️ **注意**：你**不需要**运行 `playwright install chromium`，因为程序会自动调用你电脑上已有的 Chrome 浏览器。

---

## 🚀 第四步：运行查询程序

一切准备就绪，现在可以开始自动查询了！

### 操作步骤

1. 确保命令窗口当前仍在 `证书查询` 文件夹路径下（看窗口标题或路径提示）
2. 输入以下命令，按回车：
   ```
   python sport_level_crawler.py
   ```
3. 程序会自动打开一个 Chrome 浏览器窗口，并开始访问国家体育总局网站
4. **如果网站要求登录**：程序会暂停，命令窗口里会提示你手动完成登录（扫码或输入账号密码）。登录完成后，回到命令窗口按一下 **回车键** 继续。
5. **如果出现验证码**：程序会自动把验证码图片保存到 `data` 文件夹，并在命令窗口提示你输入验证码。输入后按回车即可。
6. 程序会自动填写证书编号和姓名，点击查询，等待结果出现后保存数据。
7. 如果查询结果页面有「秩序册」或「成绩册」的下载链接，程序会自动下载并保存。

---

## 📁 第五步：查看查询结果

程序运行完毕后，在你的 `证书查询` 文件夹里会自动生成一个 `data` 文件夹，里面包含：

| 文件名格式 | 内容说明 |
|-----------|---------|
| `result_姓名_证书编号.json` | 该条记录的详细查询结果（表格数据 + 页面文本） |
| `秩序册_姓名_证书编号.pdf` | 自动下载的秩序册文件（如果有的话） |
| `成绩册_姓名_证书编号.pdf` | 自动下载的成绩册文件（如果有的话） |
| `summary_2026xxxx_xxxxxx.json` | 所有查询记录的汇总结果 |

你可以直接用「记事本」打开 `.json` 文件查看文字结果，用 Chrome 或 PDF 阅读器打开 `.pdf` 文件。

## 🆘 常见问题排查

### 1. 提示「找不到证书编号输入框」或「找不到查询按钮」

这说明网页的实际按钮位置和程序猜测的不一样。

**解决办法：**
1. 程序运行时会自动打开 Chrome，在网页上按键盘 `F12` 打开开发者工具
2. 点击开发者工具左上角的「箭头」图标，然后点击网页上的输入框
3. 在右侧代码区右键该元素 → `Copy` → `Copy selector`
4. 用记事本打开 `sport_level_crawler.py`，找到 `SELECTORS = {` 这一段
5. 把对应的选择器替换为你复制的内容，保存后重新运行

### 2. Chrome 浏览器没有自动打开

**检查：**
- 你的电脑上是否安装了 Google Chrome？（不是 360、Edge、QQ 浏览器，必须是 Chrome）
- 如果 Chrome 安装在了非默认位置，打开脚本找到 `find_local_chrome()` 函数，在里面添加你的 Chrome 实际路径，例如：
  ```python
  candidates = [
      r"C:\Program Files\Google\Chrome\Application\chrome.exe",
      r"D:\Software\Chrome\chrome.exe",  # ← 添加你的路径
  ]
  ```

### 3. 下载速度很慢或下载失败

政府网站的文件服务器有时不稳定。程序会自动尝试两种下载方式：
- 直接链接下载
- 浏览器点击下载

如果都失败了，你可以手动在打开的 Chrome 窗口里点击下载链接，文件会进入你的「下载」文件夹。

### 4. 程序运行一半报错退出了

不要慌，程序已经自动把出错时的网页截图保存到了 `data/error_xxx.png`，你可以打开图片看看当时网页显示什么，然后告诉我截图内容，我可以帮你调整。
